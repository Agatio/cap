General Information
In this section you will find information about all the new (and some old) inputs and outputs for the wire mod.

You can find some examples of e2 chips in addon folder: data\Expression2\e2shared\stargate.
In the game, they are also available in a folder e2shared\stargate when choosing what e2 chip to open.
General inputs and outputs on stargates
Wire Inputs:
  • Dial Address - dial address by old method, (for example - with wired keyboard).
  • Dial String [STRING] - for set address what you want to dial (string).
  • Dial Mode - for set dialling mode (slow/fast/nox - 0/1/2).
    Note: Orlin's gate have only fast and nox modes, supergate havn't nox dialling mode.
  • Start String Dial - start dialling with current dial string value and dial mode.
  • Close - close stargate or abort dialling.
  • Disable Autoclose - disable auto-close stargate after passing through them.
    Note: orlin gate do not have this input.
  • Refresh List - refresh list of available gates (for "Address List" output).
  • Set Point of Origin - force set point of origin for stargate (without depending is there dhd or not), 1 - earth, 2 - abydos.
    Note: stargate universe, atlantis and supergate don't have this input.
    Also for orlin and tollan stargates we can see different only on vgui menu or with e2 screens.
  • Get Energy from Address [STRING] - when you set address, then you will have in output "Enegy from Address" required amount of energy for dial this address, -1 means gate have energy from dhd/blackhole or you when havn't life support addon.
    Also this input have additional option - if add space and "D" after address (example: "SPAWN0# D"), then in output we will have distance to this gate.
  • Disable Menu - disable stargate menu (when press E).
    Note: if you disable menu, then you can't set address etc.
Wire Outputs:
  • Active - return 1 if gate activated.
  • Open - return 1 if gate opened.
  • Inbound - return 1 if gate have inbound connection.
    Note: orlin gate do not have this output.
  • Chevron - return value with number of activated chevrons.
    Note: supergate do not have this output.
  • Chevron Locked - return 1 if this is last chevron and it locked.
    Note: supergate do not have this output.
  • Chevrons [STRING] - return string with status of all chevrons (activated or not), this field should be used for a visual indicator of chevrons activation with e2.
    Note: Supergate, orlin and universe stargates don't have this output.
  • Address List [ARRAY] - return list of all available addresses from this gate for dialling (in array, not include private gates).
  • Earth Point of Origin - return 1 if will be used earth point of origin (for e2 screens).
    Note: stargate universe, atlantis and supergate don't have this output.
  • Dialing Address [STRING] - return current dialling (outbound) or dialled (inbound) address.
  • Dialing Mode - return dialling mode (not active/slow/fast/nox - -1/0/1/2). inbound always shows as fast.
  • Dialing Symbol [STRING] - return current dialling symbol, only has an effect in outbound dialling.
    Note: Supergate and orlin stargates don't have this output.
  • Dialed Symbol [STRING] - return current dialled symbol (what locked on chevron).
    Note: Supergate and orlin stargates don't have this output.
  • StarGate Address [STRING] - return current stargate address.
  • StarGate Group [STRING] - return current stargate group. Only with stargate_group_system = 1.
    Note: Universe stargate do not have this output.
  • StarGate Name [STRING] - return current stargate name.
  • StarGate Private - return current stargate private status.
  • StarGate Local - return current stargate local status. Only with stargate_group_system = 1.
  • StarGate Galaxy - return current stargate galaxy status. Only with stargate_group_system = 0.
    Note: Universe stargate do not have this output.
  • StarGate Blocked - return 1 if address is blocked.
    Note: supergate do not have this output.
  • Energy from Address - return required amount of energy from address what you set in input "Get Energy from Address", -1 means gate have energy from dhd/blackhole or you when havn't life support addon.
    Note: Output "Energy from Address" return amount of energy for open stargate, not for keep connection open, also can ouput distance to gate if added option "D" in input.
Additional wire inputs and outputs - Stargate SG1/Infinity
Additional wire inputs:
  • Rotate Ring - run the rotation of the ring.
  • Ring Speed Mode - set ring speed mode, 0 (default) = 50%, -1 = 25%, 1 = 100% (normal), 2 = 150%, 3 = 200%.
  • Chevron Encode - Encode chevron with current symbol on ring.
  • Chevron 7 Lock - lock Chevron 7 with current symbol on ring and open gate if all correct.
  • Activate chevron numbers [STRING] - specify a string with the activation of all chevrons (active or not), this field can be used to activate the chevrons with e2 in various combinations.
  • SGC Type - rotate SGC Type connection, also will be chev7 lock in this case, and different sounds.
    Note: this input can be saved with gatespawner.
Additional wire outputs:
  • Ring Symbol [STRING] - show currect symbol letter (what on chevron 7).
  • Ring Rotation - show ring symbol rotation status, 0 - idle, 1 - clockwise, -1 - counter-clockwise rotation.
Additional wire inputs and outputs - Stargate Movie
Additional wire inputs:
  • Chevron Light - enable chevron light when stargate activated.
  • Classic Mode - enable classic mode for dialling scene.
  • Rotate Ring - run the rotation of the ring.
  • Ring Speed Mode - set ring speed mode, 0 (default) = 50%, -1 = 25%, 1 = 100% (normal), 2 = 150%, 3 = 200%.
  • Chevron Encode - Encode chevron with current symbol on ring.
  • Chevrons Lock - Lonk all chevrons with current symbol on ring (must take the value from the output "Ring Chev7 Symbol [STRING]") and open gate if all correct.
  • Activate chevron numbers [STRING] - specify a string with the activation of all chevrons (active or not), this field can be used to activate the chevrons with e2 in various combinations.
  • SGC Type - rotate ring on inbound connection, also will be chev7 lock in this case and different sounds.
    Note: inputs "Chevron Light", "Classic Mode" and "SGC Type" can be saved with gatespawner.
Additional wire outputs:
  • Ring Symbol [STRING] - show currect symbol letter on chevron what need encode.
  • Ring Chev7 Symbol [STRING] - show currect symbol letter on chevron 7 (top).
  • Ring Rotation - show ring symbol rotation status, 0 - idle, 1 - clockwise, -1 - counter-clockwise rotation.
Additional wire inputs and outputs - Stargate Atlantis
Additional wire inputs:
  • Turn on ring light - turning on all symbols (light) on the ring.
  • Activate chevron numbers [STRING] - specify a string with the activation of all chevrons (active or not), this field can be used to activate the chevrons with e2 in various combinations.
Additional wire inputs and outputs - Stargate Universe
Additional wire inputs:
  • Rotate Ring - run the rotation of the ring.
  • Ring Speed Mode - set ring speed mode, 0 (default) = 50%, 1 = 100% (normal).
  • Encode Symbol - Encode currect symbol on top of ring.
  • Symbols Lock - Encode last symbol and open gate if all correct.
  • Inbound Symbols - enable symbols light in inbound connection. 1 - dialled symbols, 2 - all symbols.
  • Activate chevrons - activate all chevrons light (from 1 to 9).
  • Activate Symbols - activate all symbols (light).
Additional wire outputs:
  • Ring Symbol [STRING] - show currect symbol letter (what on top).
  • Ring Rotation - show ring symbol rotation status, 0 - idle, 1 - clockwise, -1 - counter-clockwise rotation.
  • StarGate Type [STRING] - return current stargate type. Only with stargate_group_system = 1.
Additional wire inputs and outputs - Tollan Stargate
Additional wire inputs:
  • Activate chevron numbers [STRING] - specify a string with the activation of all chevrons (active or not), this field can be used to activate the chevrons with e2 in various combinations.
Wire inputs and outputs on Stargate Iris Computer
Wire Inputs:
  • Iris Control - activate or deactivate iris.
  • GDO Status - transmit to gdo that code is valid (for e2 chips).
  • GDO Text [STRING] - transmit text to gdo (for e2 chips).
  • Auto-close - automatically close iris on incoming connections.
  • Don't Auto-Open - disable automatic opening of the iris when entered correct code.
  • Close time - automatically close the iris through a specified number of seconds after entering the correct code.
Wire Outputs:
  • Incoming Wormhole - returns 1 if this is incoming connection.
  • Code Status - return 1 if code is correct.
  • Gate Active - return 1 if gate active.
  • Received Code - return code what transmitted from gdo through stargate.
  • Code Description [STRING] - return description of transmitted code.
  • Iris Active - return 1 if iris activated.
Wire inputs and outputs on Stargate Iris
Wire Inputs:
  • Activate - if 1 - activate, 0 - deactivate.
  • Toggle - activate or deactivate iris.
Wire Outputs:
  • Activated - return 1 if iris activated.
Wire inputs and outputs on DHD's
Wire Inputs:
  • Press Button - press button on dhd (for example - with wired keyboard), use "*" for random address.
  • Disable Menu - disable dhd menu (vgui), so you can dial only by pressing buttons.
  • Disable Ring Rotation - disable ring rotation with on-button-press feature on dhd's.
    Note: this input exists only on sg1/infinity/concept dhd's. Can be saved with gatespawner.
  • Slow Mode - enables slow dial mode with on-button-press feature on dhd.
    Note: this input exists only on atlantis/city dhd's. Can be saved with gatespawner.
