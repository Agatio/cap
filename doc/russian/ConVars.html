Сохранение настроек на сервере:
Для сохранения настроек на сервере, пропишите название переменной и её значение в файл "garrysmos\cfg\server.cfg".
Пример: stargate_candial_groups_menu 1
Серверные настройки (переменные):
  • stargate_group_system <1/0> - если установить 0, тогда будет использоваться галактическая система набора врат из CAP.
    Примечание: после изменения переменной "stargate_group_system" вы должны подождать 5-10 секунд, пока система врат перезагрузится.
  • stargate_candial_groups_dhd <1/0> - разрешит набор адресов больше 7-ми символов на наборном устройстве (по умолчанию 1). Включая мобильные наборные устройста (как например на джампере и тд). Не влияет на консоли атлантиса и вселенной.
  • stargate_candial_groups_menu <1/0> - разрешит набор адресов больше 7-ми символов из меню звёздных врат (по умолчанию 1).
  • stargate_candial_groups_wire <1/0> - разрешить набор адресов больше 7-ми стмволов с помощью wire (по умолчанию 1). Включая ручной режим медленного набора wire.
  • stargate_sgu_find_range <16000> - дистанция поиска sgu (по умолчанию 16000). Если universe врата очень далеко и тойже группы - тогда вы можете набрать их только используя 8 шевронов. 0 - выключить (разрешить всегда набирать используя 7 шевронов).
  • stargate_energy_dial <1/0> - включить потребление энергии звёздными вратами.
  • stargate_energy_dial_spawner <0/1> - если установить в 1 (по умолчанию 0), тогда врата спавненые gatespawner'ом (сервером) нуждаются в энергии тоже.
  • stargate_different_dial_menu <0/1> - вы можете набирать врата только с наборного устройства (включая мобильные наборные устройства).
  • stargate_dhd_protect <0/1> - если установить в 1 (по умолчанию 0), тогда игроки не смогут уничтожать наборные устройства.
  • stargate_dhd_protect_spawner <0/1> - если установить в 1 (по умолчанию 0), тогда игроки не смогут уничтожать наборные устройства созданные gatespawner'ом (сервером).
  • stargate_dhd_destroyed_energy <1/0> - если установить в 0 (по умолчанию 1), тогда уничтоженные наборные устройства не дают энергию для врат.
  • stargate_dhd_close_incoming <1/0> - если установить в 0 (по умолчанию 1), тогда нельзя будет закрывать входящие соеденения при помощи наборного устройства (dhd).
  • stargate_show_inbound_address <2/1/0> - по умолчанию 2, 0 - не показывать входящие адреса на dhd/wire, 1 - не показывать только если врата приватные, 2 - показывать всегда.
  • stargate_protect <0/1> - если установить в 1 (по умолчанию 0), тогда игроки не смогут уничтожать шевроны звёздных врат.
  • stargate_protect_spawner <0/1> - если установить в 1 (по умолчанию 0), тогда игроки не смогут уничтожать шевроны звёздных врат созданные gatespawner'ом (сервером).
  • stargate_block_address <2/1/0> - по умолчанию 2, фукнция блокировки адресов доступна для всех врат, если установить в 1, тогда функция будет доступна только для звёздных врат вселенной, установите 0 для полного отключения функции.
  • stargate_dhd_letters <1/0> - по умолчанию 1, показывает буквы на наборном устройстве.
  • stargate_energy_target <1/0> - по умолчанию 1, включает потребление энергии с врат которые вы набрали.
  • stargate_vgui_glyphs <2/1/0> - показывать символы и буквы в меню звёздных врат, 2 - оба поля (по-умолчанию), 1 - только символы, 0 - только буквы.
  • stargate_dhd_menu <1/0> - показ меню при нажатии на центральную кнопку наборного устройства (по-умолчанию 1).
  • stargate_atlantis_override <1/0> - включить возможность переопределения sg1-типа врат, вратами атлантиса (по-умолчанию 1).
  • stargate_gatespawner_enabled <1/0> - включить gatespawner (по-умолчанию 1).
    Примечание: необходимо прописать stargate_reload для того чтобы изменения вступили в силу.
  • stargate_dhd_ring <1/0> - включить вращение кольца при нажатии кнопок на dhd (по-умолчанию 1).

    • Подсказка: Напишите в консоли stargate_system_convars для открытия меню редактирования переменных!
