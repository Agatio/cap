Общая информация
В данном разделе вы найдёте информацию обо всех новых (и некоторых старых) входах и выходах для wire mod.

Вы можете найти несколько примеров e2 скриптов в папке аддона: data\Expression2\e2shared\stargate.
В игре они также доступны в папке e2shared\stargate при выборе какой e2 чип открыть.
Основные входы и выходы у врат
Wire Входы:
  • Dial Address - набрать адресс старым методом (например с помощью wired keyboard).
  • Dial String [STRING] - установить адрес, который нужно набрать (string).
  • Dial Mode - установить режим набора (меденный/быстрый/ноксовый - 0/1/2).
    Примечание: Orlin врата имеют только быстрый и ноксовый набор, супер врата не имеют ноксовый набор.
  • Start String Dial - начать набор с установленым адресом (string) и режимом.
  • Close - закрыть врата или остановить набор.
  • Disable Autoclose - выключить автоматическое закрытие врат после прохождение через них.
    Примечание: Orlin врата не имеют этого входа.
  • Refresh List - обновить список доступных врат (для выхода "Address List").
  • Set Point of Origin - принудительно установить точку отправления врат (вне зависимости наличия dhd), 1 - земная, 2 - абидос.
    Примечание: Этого входа нету у звёздных врат вселенной, атлантис и суперврат.
    А также у врат орлина и толлан увидеть разницу можно только в наборном меню или на e2 экранах.
  • Get Energy from Address [STRING] - когда задан адресс, тогда вы получите на выходе "Enegy from Address" необходимое количество энергии для набора этого адресса, -1 означает что вы имеете энергию от dhd/blackhole или у вас нету life support аддона.
    А также данный вход имеет дополнительные параметр - если добавить после адреса пробел и "D" (например: "SPAWN0# D"), то мы получим дистанцию к этим вратам на выходе.
  • Disable Menu - отключить меню звёздных врат (при нажатии на E).
    Примечание: если отключить меню, тогда нельзя надать адресс и тд у врат.
Wire Выходы:
  • Active - возвращает 1 если врата активны.
  • Open - возвращает 1 если врата открыты.
  • Inbound - возвращает 1 если это входящее соеденение.
    Примечание: Orlin врата не имеют этого выхода.
  • Chevron - возвращает значение с числом активированных шевронов.
    Примечание: Этого выхода нету у суперврат.
  • Chevron Locked - возвращает 1 если это последний шеврон и он заблокирован.
    Примечание: Этого выхода нету у суперврат.
  • Chevrons [STRING] - возвращает строку со статусом для всех шевронов (активен или нет), это поле например можно использовать для визуального индикатора активации шевронов с помощью e2.
    Примечание: Этого выхода нету у supergate, orlin и universe врат.
  • Address List [ARRAY] - возвращает список всех доступных адресов для набора с этих врат (массив, не включает приватные адреса).
  • Earth Point of Origin - возвращает 1 если будет использована земная точка отправления (для e2 скринов).
    Примечание: Этого выхода нету у звёздных врат вселенной, атлантис и суперврат.
  • Dialing Address [STRING] - возвращает набираемый адрес (исходящие соеденение) или набранный адрес (входящее соеденение).
  • Dialing Mode - возвращает режим набора (не активно/медленный/быстрый/нокс - -1/0/1/2). входящие соеденения всегда показываются как быстрый набор.
  • Dialing Symbol [STRING] - возвращает набираемый символ, имеет эффект только для исходящего соеденения.
    Примечание: Этого выхода нету у суперврат и врат orlin'а.
  • Dialed Symbol [STRING] - возвращает набранный символ (что заблокировался на шевроне).
    Примечание: Этого выхода нету у суперврат и врат orlin'а.
  • StarGate Address [STRING] - возвращает адрес врат.
  • StarGate Group [STRING] - возвращает группу врат. Только при stargate_group_system = 1.
    Примечание: Этого выхода нету у звёздных врат вселенной.
  • StarGate Name [STRING] - возвращает имя врат.
  • StarGate Private - возвращает приватный статус врат.
  • StarGate Local - возвращает локальный статус врат. Только при stargate_group_system = 1.
  • StarGate Galaxy - возвращает галактический статус врат. Только при stargate_group_system = 0.
    Примечание: Этого выхода нету у звёздных врат вселенной.
  • StarGate Blocked - возвращает 1 если адрес врат заблокирован.
    Примечание: Этого выхода нету у суперврат.
  • Energy from Address - возвращает количество энергии необходимые для набора адресса из входа "Get Energy from Address", -1 означает что вы имеете энергию от dhd/blackhole или у вас нету life support аддона.
    Причечание: Выход "Energy from Address" выдаёт количество энергии для открытия врат, не для поддержания связи в дальнейшем, а также может выводит дистанцию к вратам, если добавлена опция "D" на входе.
Дополнительные входы и выходы - Stargate SG1/Infinity
Дополнительные wire входы:
  • Rotate Ring - вращать кольцо.
  • Ring Speed Mode - установить скорость вращения кольца, 0 (по умолчанию) = 50%, -1 = 25%, 1 = 100% (нормальная), 2 = 150%, 3 = 200%.
  • Chevron Encode - Закодировать шеврон с текущим символом на кольце.
  • Chevron 7 Lock - Заблокировать шеврон 7 с текущим символом на кольце, и если всё верно - открыть врата.
  • Activate chevron numbers [STRING] - задать строку с активацией для всех шевронов (активен или нет), это поле можно использовать например для активации шевронов с помощью e2 в разных комбинациях.
  • SGC Type - Вращать кольцо при входящем соеденении, а также будет анимация блокировки 7 шеврона, и другие звуки.
    Причечание: этот вход может быть сохранён с gatespawner.
Дополнительные wire выходы:
  • Ring Symbol [STRING] - показывает текущую букву символа на кольце (что на 7 шевроне).
  • Ring Rotation - показывает статус вращения кольца врат, 0 - не вращается, 1 - вращение по часовой стрелке, -1 - вращение притив часовой стрелки.

  • Activate chevron numbers [STRING] - задать строку с активацией для всех шевронов (активен или нет), это поле можно использовать например для активации шевронов с помощью e2 в разных комбинациях.

    Причечание: входы "Chevron Light", "Classic Mode" и "SGC Type" могут быть сохранены с gatespawner.
Дополнительные wire выходы:
  • Ring Symbol [STRING] - показывает текущую букву символа на кольце в зависимости от набираемого шеврона.
  • Ring Chev7 Symbol [STRING] - показывает текущую букву символа на кольце что на 7 шевроне.
  • Ring Rotation - показывает статус вращения кольца врат, 0 - не вращается, 1 - вращение по часовой стрелке, -1 - вращение притив часовой стрелки.

Входы и выходы у Stargate Iris Computer
Wire Входы:
  • Iris Control - активировать или деактивировать iris.
  • GDO Status - передать на gdo что код верный (для e2 чипов).
  • GDO Text [STRING] - передать указаный текст на gdo (для e2 чипов).
  • Auto-close - автоматически закрывать iris при входящем соединении.
  • Don't Auto-Open - запретить автоматическое открытие iris при верном коде.
  • Close time - автоматически закрыть iris через указанное количество секунд после введения верного кода.
Wire Выходы:
  • Incoming Wormhole - возвращает 1 если это входящее соединение.
  • Code Status - возвращает 1 если введённый код верный.
  • Gate Active - возвращает 1 если врата активны.
  • Received Code - возвращает код переданный с gdo через врата.
  • Code Description [STRING] - возвращает описание введённого кода.
  • Iris Active - возвращает 1 если iris активировано.
Входы и выходы у Stargate Iris
Wire Входы:
  • Activate - если 1 - активировать, 0 - деактивировать.
  • Toggle - активировать или деактивировать iris.
Wire Выходы:
  • Activated - возвращает 1 если iris активировано.
Входы и выходы на наборных устройствах (DHD)
Wire Входы:
  • Press Button - нажать кнопку на наборном устройсте (например с помощью wired keyboard), используйте "*" для случайного адреса.
  • Disable Menu - выключить визуальное меню наборного устройства, таким образом набрать адрес можно только с помощью кнопок.
  • Disable Ring Rotation - выключить вращение кольца при нажатии на кнопках у dhd.
    Примечание: этот вход есть только у sg1/infinity/concept наборных устройств (dhd's). Может быть сохранён с gatespawner.
  • Slow Mode - включает медленый режим набора для наборного устроства при нажатии на кнопку на нём.
    Примечание: этот вход есть только у atlantis/city наборных устройств (dhd's). Может быть сохранён с gatespawner.
