Want translate stargate vgui menus to your language?
If you want translate stargate vgui menus (when you press E on stargate or dhd, etc) or some other things to your language, then read the text below.
How to translate stargate vgui menus:
  1. Go to cap\lua\data\language folder.
  2. Make copy of folder "english" and set name of copied folder with your language name.
  3. Open this folder, and then open files from this folder in editor what support utf8 charset.
  4. Translate all words in the second column.
  5. Save file in utf8 charset (if you use notepad++ then you also should convert all symbols to utf8).
  6. Run gmod, and load any map.
  7. Spawn gate and dhd, and test in-game your language file for errors and working ability.
    Also please check - whether all words fits in stargate vgui menus, convars edit menu, etc.
  8. If all works fine - put a link to download your files in this thread - http://sg-carterpack.com/index.php/topic,91.0.html.
  9. If all fine - i will add it into svn.
Attention! Be sure to check in-game! There are limits on the number of characters in the fields!
Also you can translate not all files, if you something not translated - then will be used english message.

Thank you!
Some Useful console commands for debuging:
  • cl_language - set gmod language (but it will not be saved, you should change language in steam settings).
  • sg_language - set stargate language, after that you should call sg_language_reload.
  • CAP_Outdated - display updater check message.
  • stargate_cap_convars - open cap convars/limits menu (for debugging).
  • stargate_system_convars - open stargate convars menu (for debugging).
  • stargate_cl_language_debug 1 - no need restart map for see changes in stargate vgui menus.
  • sg_language_reload - reload language files.
